### 得意な技術スタック
- **言語**： Python
- **AI / LLM**： Gemini等の各種 API (自然言語による指示、構造化データ JSON / YAML の抽出・パース)
- **データ可視化・Web**： Dash (Plotly), Cytoscape, HTML / CSS
- **その他**： 複雑なネットワーク構造・相関図の動的描画アルゴリズムの実装

### 本プロダクト（ポートフォリオ）の技術的特徴
1. **生成AIによる非構造化データのデータ化**  
   Gemini APIを用いて、ユーザの説明文や、小説や漫画などの複雑なテキストデータから、人物間の関係性を自動で抽出し構造化データ（YAML）に変換しています。
2. **独自開発した整列アルゴリズムと配線アルゴリズム**  
   人物（ノード）を男系で横に揃えたのち、それらの関係をできるだけ交差が少なくなるように配線（エッジ）しています。
   配置に使用したアルゴリズムは別のリポジトリ [SCH-MSA](https://github.com/ferveuretzen/msa-under-strict-column-homogeneity) で公開していますが、作成者ならではの凝った使い方をしています。
3. **Dash Cytoscapeによる高度なインタラクティブ描画**  
   抽出されたデータを瞬時にWeb上のネットワーク図として可視化。拡大・縮小だけでなく、ユーザーがドラッグして位置を調整できる柔軟なUIを実装しています。
4. **対話型（インタラクティブ）な図の修正アルゴリズム**  
   「言葉（自然言語）による修正指示」をAIが解釈し、描画されている図に対して動的に要素の「追加・修正・削除」を行う双方向システムを構築しています。
5. **生成されたネットワーク図の保存と活用**  
   JavaScriptへ変換されるので、図をWebサイトに貼り付けたり、ファイルとして配布したりと、自由に活用できます。（デモサイトをご覧ください）
